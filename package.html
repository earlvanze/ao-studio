Ongoing pricing clarity

$500/month service fee

This is the ongoing management layer after implementation. It covers Earl’s time, maintenance, improvements, and keeping the system running smoothly. Earl covers server costs inside that fee.

AI model cost is separate

The client also provides and pays for the AI model subscription. Depending on fit, that may be ChatGPT Plus or Pro, Claude Code Pro or Max, or Gemini.

Higher setup month, lower ongoing month is common

Some businesses only need the more expensive model during initial setup, around $200 for the first month, then can move closer to $20/month later depending on the work being done.

Local model option

If you want more local control, we can also set this up around local AI models. In that case, the client provides the hardware, such as a Mac Mini or another machine that fits the budget.

Plain-English cost structure

Earl covers server costs. The client covers the AI model. That keeps the pricing understandable and avoids hidden bundled fees.
